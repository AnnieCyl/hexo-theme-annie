# hexo-theme-abc

基于[hugo-theme-cactus-plus](https://github.com/nodejh/hugo-theme-cactus-plus)修改,cactus-plus是一个hugo theme

## Installation

plugin dependencies:

- hexo-renderer-pug
- hexo-wordcount

clone theme

    git clone https://github.com/fuzhouxxdong/hexo-theme-abc.git themes/abc

upgrade

    cd themes/abc
    git pull origin master
